# Revenue Radar

> Revenue Radar by DigitalFlow Consulting. A two-agent B2B account intelligence and outreach SaaS for founders at $1M to $25M.

## Quick start

```bash
npm install
cp .env.example .env.local   # fill in keys
npm run dev                  # http://localhost:3000
```

Build plan: [`docs/BUILD_PLAN.md`](docs/BUILD_PLAN.md). Engineering context: [`CLAUDE.md`](CLAUDE.md).

## Stack

Next.js 16 App Router, TypeScript, Tailwind v4, Supabase, Inngest, Stripe, Resend, Sentry, Posthog, Anthropic TypeScript SDK, HubSpot.
